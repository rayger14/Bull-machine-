================================================================================
SYSTEM FLOW
================================================================================

1. INITIALIZATION (Backtest Startup)
   ┌─────────────────────────────────────────────────────────────┐
   │ registry = get_registry()                                   │
   │ ├─ Loads archetype_registry.yaml                            │
   │ ├─ Validates all class paths                                │
   │ ├─ Imports archetype classes                                │
   │ └─ Verifies BaseArchetype compliance                        │
   └─────────────────────────────────────────────────────────────┘

2. FILTERING (Select Archetypes)
   ┌─────────────────────────────────────────────────────────────┐
   │ archetypes = registry.get_archetypes(                       │
   │     maturity=['production', 'calibrated'],                  │
   │     enabled_only=True,                                      │
   │     config=config                                           │
   │ )                                                           │
   │ ├─ Filters by maturity level                                │
   │ ├─ Filters by enable flags in config                        │
   │ └─ Returns: [S1, S4, S5, H, B, K]                           │
   └─────────────────────────────────────────────────────────────┘

3. FEATURE VALIDATION (Pre-Backtest)
   ┌─────────────────────────────────────────────────────────────┐
   │ gate = FeatureRealityGate()                                 │
   │ report = gate.validate_all(archetypes, df)                  │
   │ ├─ Checks critical features exist in df                     │
   │ ├─ Warns on missing recommended features                    │
   │ ├─ Reports coverage % per archetype                         │
   │ └─ Fails fast if critical features missing                  │
   └─────────────────────────────────────────────────────────────┘

4. INSTANTIATION (Create Instances)
   ┌─────────────────────────────────────────────────────────────┐
   │ instances = registry.instantiate_all(                       │
   │     maturity=['production'],                                │
   │     enabled_only=True,                                      │
   │     config=config                                           │
   │ )                                                           │
   │ ├─ Creates archetype instance for each                      │
   │ ├─ Skips stub archetypes (no implementation)                │
   │ └─ Returns: {S1: LiquidityVacuumArchetype(), ...}           │
   └─────────────────────────────────────────────────────────────┘

5. EXECUTION (Backtest Loop)
   ┌─────────────────────────────────────────────────────────────┐
   │ for idx, row in df.iterrows():                              │
   │     context = RuntimeContext(...)                           │
   │                                                             │
   │     for arch_id, arch in instances.items():                 │
   │         # VETO CHECK                                        │
   │         veto = arch.veto(context)                           │
   │         if veto.is_vetoed:                                  │
   │             continue  # Skip this archetype                 │
   │                                                             │
   │         # PATTERN SCORING                                   │
   │         score = arch.score(context)                         │
   │         if score.total_score < threshold:                   │
   │             continue  # Score too low                       │
   │                                                             │
   │         # ENTRY GENERATION                                  │
   │         entry = arch.entry(context)                         │
   │         # Execute trade...                                  │
   └─────────────────────────────────────────────────────────────┘

================================================================================
KEY INTERFACES
================================================================================

BaseArchetype (ABC)
├── Class Attributes
│   ├── ARCHETYPE_ID: str
│   ├── ARCHETYPE_NAME: str
│   ├── MATURITY: MaturityLevel
│   ├── DIRECTION: SignalType
│   ├── REGIME_TAGS: List[str]
│   └── REQUIRES_ENGINES: List[str]
│
├── Required Methods
│   ├── required_features() → List[str]
│   ├── score(context) → ArchetypeScore
│   ├── veto(context) → ArchetypeVeto
│   └── entry(context) → ArchetypeEntry
│
├── Optional Methods
│   ├── exit(context) → Optional[Dict]
│   └── diagnostics(context) → Dict
│
└── Utility Methods
    ├── get_threshold(context, param, default)
    └── validate_features(context)

ArchetypeScore
├── total_score: float [0.0, 1.0]
├── component_scores: Dict[str, float]
├── reasons: List[str]
└── metadata: Dict

ArchetypeVeto
├── is_vetoed: bool
├── reason: str
└── veto_type: str

ArchetypeEntry
├── signal: SignalType (LONG/SHORT/FLAT)
├── confidence: float [0.0, 1.0]
├── entry_price: Optional[float]
└── metadata: Dict (stop_loss, take_profit, etc.)

================================================================================
FEATURE CLASSIFICATION
================================================================================

CRITICAL (Must have - blocks execution if missing)
├── Example: liquidity_score, volume_zscore, wick_lower_ratio
└── Behavior: FeatureRealityGate.validate_all() fails if missing

RECOMMENDED (Should have - degraded mode without)
├── Example: VIX_Z, DXY_Z, funding_Z
└── Behavior: Warning logged, archetype runs in degraded mode

OPTIONAL (Nice to have - graceful fallback)
├── Example: crisis_composite, capitulation_depth
└── Behavior: Silent fallback to default values

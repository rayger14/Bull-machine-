HOW TO USE
================================================================================

1. Feature Engineering Script:

   python3 bin/engineer_all_features.py

   Input: data/features_2018_2024_complete.parquet (235 features)
   Output: data/features_2018_2024_complete.parquet (241 features)

2. Load Model:

   import pickle
   with open('models/continuous_risk_score_v1.pkl', 'rb') as f:
       model_data = pickle.load(f)
   model = model_data['model']
   features = model_data['features']

3. Predict Risk Score:

   X = df[features].ffill().bfill().fillna(0)
   risk_score = model.predict(X)[0]  # 0.0-1.0

4. Discretize to Regime:

   if risk_score < 0.25:
       regime = 'crisis'
   elif risk_score < 0.45:
       regime = 'risk_off'
   elif risk_score < 0.65:
       regime = 'neutral'
   else:
       regime = 'risk_on'

# Archived Reports

Historical experiment reports and optimization studies from the Bull Machine trading system.

## Structure

- **`archetype_optimization/`** - Archetype parameter tuning studies and validation
- **`exit_optimization/`** - Exit strategy optimization experiments
- **`frontier_exploration/`** - Bull/bear frontier exploration studies
- **`baseline_studies/`** - Baseline performance studies (2024, v10)
- **`experiments/`** - Various experimental runs and studies
  - BTC/ETH asset-specific experiments
  - Optuna hyperparameter optimization runs
  - Router validation studies
  - Multi-asset comparisons
  - What-if scenario analyses
- **`legacy/`** - Legacy reports from earlier versions

## Note

These reports are archived for historical reference. For current performance metrics and analysis, see the active reports in `/docs/reports/`.
